You can add new executables in this directory, adding to file "list.drl_<Program>" the executable path, example:

To list.drl:
if is C/C++ based the program: ./usr/share/diamond-ring/your_executable.***
if is script (Only Shell/Bash): . /usr/share/diamond-ring/your_script.***
if is other (Make folder for you script/executable): PATH=$PATH:/usr/share/diamond-ring/others; your_program.***
